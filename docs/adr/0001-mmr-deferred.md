# ADR-0001 — MMR 시스템 구현 유보

## 상태

보류 (향후 구현 예정)

## 결정

MMR을 현재 구현하지 않는다. 티어리스트와 정렬은 승률 기반으로만 운영한다.

## 이유

- 내전 초기 단계에서 Elo 계산에 필요한 경기 샘플 수가 부족함
- 승률만으로도 티어 구분이 충분히 의미 있음
- 구현 복잡도 대비 초기 효용이 낮음

## 향후 구현 시 고려사항

- Elo 방식 채택 시 `Streamer`에 `mmr: number` 필드 추가
- 경기 결과 저장 시 양 팀 MMR 평균 기반으로 증감 계산
- 기존 경기 기록 소급 적용 여부 결정 필요
- MMR 표시 위치: 티어리스트 플레이어 셀, 개인 전적 사이드바

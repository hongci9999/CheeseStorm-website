# CONTEXT — Cheesestorm

치지직 스트리머들의 HotS 내전 결과를 기록하고 티어리스트를 보는 플랫폼.

## 용어 사전

### 내전 (In-house match)
스트리머 커뮤니티 내부에서 진행하는 팀 대전. 외부 래더 경기와 구분됨. 이 플랫폼이 기록하는 유일한 경기 유형.

### 스트리머 (Streamer)
내전에 참가하는 플레이어. `name`, `chzzkId`, `role` 보유. 치지직 플랫폼 소속임을 전제.

### 롤 (Role)
스트리머의 주 포지션. 값: `탱커` · `투사` · `암살자` · `지원가` · `전문가`. 스트리머 등록 시 지정, 변경 가능.

### 영웅 (Hero)
HotS 게임 내 캐릭터. 각 내전에서 스트리머가 플레이한 영웅을 경기 기록에 함께 저장함.

### 경기 (Match)
한 번의 내전. 블루팀과 레드팀 각 5인, 승자, 맵, 경기시간 포함. 각 팀원은 `[스트리머명, 영웅명]` 쌍으로 기록.

### 진영 (Side)
블루팀(`blue`) 또는 레드팀(`red`). 경기 내 팀 구분 단위.

### 티어 (Tier)
승률 기반 등급. S / A / B / C / D / unranked(미집계). 최소 3경기 충족 시 부여.

| 티어 | 승률 | 최소 경기 |
|------|------|---------|
| S    | 65%+ | 3경기   |
| A    | 55–65% | 3경기 |
| B    | 45–55% | 3경기 |
| C    | 35–45% | 3경기 |
| D    | ~35% | 3경기   |
| unranked | — | 3경기 미만 |

### 전적 (Record)
스트리머 개인의 누적 승/패 통계. 승률, 티어, 연승/연패 포함.

### 연승 / 연패 (Win streak / Loss streak)
가장 최근 경기부터 연속된 승 또는 패 수. 경기 기록에서 실시간 계산.

### MMR
현재 미구현. 향후 Elo 방식 추가 예정 (ADR-0001 참조).

## 페이지 구조

- `/` — 티어리스트 (스트리머 전체 티어 현황)
- `/matches` — 경기 목록 (타임라인 피드)
- `/matches/new` — 경기 등록
- `/streamers` — 스트리머 목록 + 추가/삭제
- `/streamers/[id]` — 개인 전적 페이지 (영웅 통계, 최근 매치)

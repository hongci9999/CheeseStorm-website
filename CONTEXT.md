# CONTEXT — Cheesestorm

치지직 스트리머들의 HotS 내전 결과를 기록하고 티어리스트를 보는 플랫폼.

## 용어 사전

### 내전 (In-house match)

스트리머 커뮤니티 내부에서 진행하는 팀 대전. 외부 래더 경기와 구분됨. 이 플랫폼이 기록하는 유일한 경기 유형.

### 스트리머 (Streamer)

내전에 참가하는 플레이어. `name`(표시명), `chzzkId`, `role`, `gameNames`(인게임 이름 목록) 보유. 치지직 플랫폼 소속임을 전제.

### 인게임 이름 (Game name)

HotS 스코어 화면에서 영웅 아이콘 아래 표시되는 배틀태그. 스트리머의 표시명·`chzzkId`와 별개의 식별자이며, 부캐·닉변으로 한 스트리머가 여럿 가질 수 있어 `gameNames` 배열로 보관. 스크린샷에서 추출한 이름을 스트리머에 매칭하는 기준이 된다.

### 미매칭 (Unmatched)

스크린샷에서 추출한 인게임 이름이 어느 스트리머의 `gameNames`와도 일치하지 않은 상태. 입력자가 수동으로 스트리머를 지정하면 그 이름이 해당 스트리머의 `gameNames`에 추가되어 다음부터 자동 매칭된다(자가 학습).

### 롤 (Role)

스트리머의 주 포지션. 값: `탱커` · `투사` · `암살자` · `지원가` · `전문가`. 스트리머 내전 기록에 따라 반영, 수동 변경 가능.

### 영웅 (Hero)

HotS 게임 내 캐릭터. 각 내전에서 스트리머가 플레이한 영웅을 경기 기록에 함께 저장함.

### 경기 (Match)

한 번의 내전. 블루팀과 레드팀 각 5인, 승자, 맵, 경기시간 포함. 각 팀원은 `[스트리머명, 영웅명]` 쌍으로 기록. 같은 멤버 구성으로 하루에 여러 번 치러질 수 있어, 경기는 (날짜 + 멤버)만으로 식별되지 않는다 — 경기시간(`dur`)이 사실상의 구분자.

### 팀 (Team: 블루 / 레드)

한 경기의 두 팀을 가르는 **임의 레이블**. `blue` · `red`는 인게임 진영(좌/우 스폰)과 무관하며 그 자체로는 아무 의미도 갖지 않는다 — 오직 "누가 같은 팀이었나"와 "어느 팀이 이겼나"만 의미가 있다. 관례상 스크린샷을 업로드한 사람의 팀이 `blue`로 들어가지만, 어떤 기능도 이 관례에 의존하지 않는다. (ADR-0002 참조)
_Avoid_: 레이블을 인게임 사이드로 해석하는 것

### 인게임 진영 (In-game side, 선택)

실제 게임 내 좌(블루)·우(레드) 스폰. 팀 레이블에서 **파생되지 않고**, 알 때만 `Match.leftTeam?`(어느 레이블 팀이 좌측 스폰이었나)에 수동 기록하는 선택 필드. 맵별 진영 승률은 이 값이 채워진 경기에 한해 산출한다.

### 경기 스탯 (Match stat)

한 경기에서 플레이어 개인의 수치 기록 — 킬·어시·데스, 공성딜, 영웅딜, 힐량, 자가힐, 경험치기여. 스크린샷에서만 추출되고 스크린샷은 경기 후 소실(재취득 불가)되므로 입력 시점에 함께 저장한다. **읽는 화면이 없어 보여도 죽은 코드가 아니다** — 보존성 데이터이며 경기 상세·개인 전적에서 노출 예정.

### 티어 (Tier)

승률 기반 등급. S / A / B / C / D / unranked(미집계). 최소 3경기 충족 시 부여.

| 티어     | 승률   | 최소 경기  |
| -------- | ------ | ---------- |
| S        | 65%+   | 3경기      |
| A        | 55–65% | 3경기      |
| B        | 45–55% | 3경기      |
| C        | 35–45% | 3경기      |
| D        | ~35%   | 3경기      |
| unranked | —      | 3경기 미만 |

### 전적 (Record)

스트리머 개인의 영웅폭,누적 승/패 통계. 승률, 맵별 승률, 같은 팀이면 승률이 잘나오는 스트리머, 적으로 만나면 불리한 스트리머, 연승/연패 포함.

### 연승 / 연패 (Win streak / Loss streak)

가장 최근 경기부터 연속된 승 또는 패 수. 경기 기록에서 실시간 계산.

### MMR

현재 미구현. 향후 Elo 방식 추가 예정 (ADR-0001 참조).

## 페이지 구조

- `/` — 티어리스트 (스트리머 전체 티어 현황)
- `/matches` — 경기 목록 (타임라인 피드)
- `/matches/new` — 경기 등록
- `/streamers` — 스트리머 목록 + 추가/삭제
- `/streamers/[id]` — 개인 전적 페이지 (영웅 통계, 최근 매치)
